# 杜威《我们如何思维》深层内容参考

## 完整思维五步法 — 三个经典案例（第六章）

### 案例一：赶赴约会（日常生活）
- **困难**：16街看到时钟12:20，但1:00需要在124街赴约
- **定义**：从当前位置如何准时到达目的地
- **假说**：地面车辆 vs 地铁 vs 高架铁路
- **推理**：地车会迟到20分钟；地铁更快但需找车站；高架铁路也有车站问题
- **验证**：最终选择地铁，1:00准时到达

### 案例二：渡船横杆（观察推理）
- **困难**：船头横杆看起来像旗杆，但横着+无滑轮=不像旗杆
- **假说**：旗杆？装饰品？无线电天线？航向指示杆？
- **推理**：旗杆应直立→排除；天线应在最高处→排除；装饰品→所有船都有，不成立；航向杆→位置合适+方向合适→合理
- **验证**：直接观察确认——位置在驾驶室前下方，舵手能清楚看到

### 案例三：肥皂泡实验（科学推理）
- **困难**：热肥皂水洗杯后，气泡先出现在杯口外沿，后又进到杯口里面
- **假说**：杯内空气膨胀导致
- **推理**：热杯取出→杯内进冷空气→冷空气受热膨胀→冒泡；杯子冷却→空气收缩→气泡进到杯内
- **验证**：实验对比（晃动的杯子冒泡/不晃的杯子不冒）→确认；再用冰块冷却验证后半段

## 杜威论"思维"的精确定义（第一章）

"对任何一个信念或假定的知识，均以积极的、执着的和用心的态度考虑它所依据的根据是否成立，若能成立，再考虑它所导致的进一步的结论，这就构成思考。"

关键要素：
1. **积极的** — 主动怀疑，不是被动接受
2. **执着的** — 不轻易放弃探究
3. **用心的** — 专心、不敷衍
4. **考察根据** — 证据是否充分？来源是否可靠？
5. **考察后果** — 如果这个信念是对的，会导致什么进一步的结论？

## 杜威论教育与思维训练（第四至五章）

- 不存在单独的"思维力"——思维是通过具体事物进行的
- 教师的影响超越课程本身：榜样比言辞更有力
- 外在成果（考试、分数、评比）往往压制真实思维
- "信息与智慧的区别"——信息是贮存的知识，智慧是运用知识的力量

## 分析和综合的关系（第八章）

- 分析 = 从混乱中澄清，挑出重要特征
- 综合 = 将分拆的事实放进更广泛的情境去理解
- 两者不可分割：分析导向综合，综合改善分析

## 意义的获得（第九章）

- 直接理解 — 熟悉后可以直接识别（如看到椅子就知道是椅子）
- 间接理解 — 陌生时需要推理（如看不懂的符号需要借助已知知识去解释）
- 知识的进步 = 直接理解和间接理解的有规律的循环运动
